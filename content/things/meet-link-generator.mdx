# Meet Link Generator

What if my Google Meet link spelled out actual words?

Google Meet links follow a pattern like `xxx-xxxx-xxx`. Most are random gibberish, but occasionally — purely by chance — you get something like `cool-cats-rock`.

This script automates the hunt for those links. Generate thousands, filter for English words, save the winners.

---

The success rate is roughly 1 in 500. Not great, but the script is patient. It'll run overnight if you let it.

This is what happens when a developer gets bored. Not every project needs to solve a real problem. Sometimes the fun is in the hunt itself.

---

**Tech:** Python, Selenium for browser automation, NLTK for the English dictionary.

Has an accompanying [blog post](https://dakheera47.hashnode.dev/) explaining the journey. Use responsibly — this is more curiosity than utility.
